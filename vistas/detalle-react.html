CURSO DE REACT

Valor: $60.-

Modalidad: Presencial.

Tiempo de dedicación necesario: 40 horas

Descripción del curso: Este curso proporciona una introducción a React, una de las bibliotecas de JavaScript más populares para el desarrollo de interfaces de usuario. Aprenderás los fundamentos de React, incluyendo la creación de componentes, manejo del estado y propiedades, y el uso del enrutamiento para construir aplicaciones web dinámicas. A través de ejercicios prácticos y proyectos, desarrollarás habilidades para construir aplicaciones web modernas y eficientes, mejorando la experiencia del usuario. Ideal para desarrolladores que desean comenzar a trabajar con React y crear interfaces interactivas y responsivas.

Requisitos previos:

    • Conocimientos básicos de HTML y CSS
      Familiaridad con JavasScript, incluyendo conceptos de variables, funciones y estructuras de control
      Experiencia previa en desarrollo web y manejo de herramientas básicas de programación.

    DOCENTE

    Lic. Carla Torres

    La Licenciada Carla Torres es una especialista en desarrollo frontend con una amplia experiencia en React. Se graduó en la Universidad de Buenos Aires (UBA) con una Licenciatura en Ciencias de la Computación y completó un Máster en Desarrollo de Aplicaciones Web en la Universidad Nacional de La Plata (UNLP). Carla ha trabajado en numerosos proyectos de desarrollo web utilizando React, y ha sido parte de equipos de desarrollo en empresas tecnológicas y startups innovadoras. Su experiencia incluye el uso de React para construir aplicaciones web interactivas y de alto rendimiento, así como el uso de herramientas complementarias como Redux y TypeScript. Como docente, Carla se destaca por su habilidad para enseñar de manera clara y estructurada. Ha impartido cursos y talleres en varias instituciones educativas y es conocida por su enfoque práctico y su capacidad para guiar a los estudiantes a través de proyectos reales. Su pasión por React y su compromiso con la educación se reflejan en la calidad de sus clases y en el éxito de sus alumnos.
